# Neuro208: Principles of Computational Modeling in Neural Circuits

Welcome to the official coding repository for **Neuro208**. This repository hosts all the interactive Python tutorials, computational models, and coding exercises used throughout the course.

## 🎯 Course Overview
This course bridges the gap between biological neural networks and computational modeling. Through hands-on Python coding, we will explore how single neurons and neural circuits process information, learn, and compute.

## 📂 Tutorial Directory
The tutorials are organized sequentially. Each folder contains the required Google Colab notebook (`.ipynb`), datasets, and supplementary materials.

* **`Tutorial_01/`**

## 🚀 How to Use These Tutorials
All coding in this course is designed to be executed in **Google Colab**. You do not need to install Python on your local machine.

1.  Navigate to the specific tutorial folder above.
2.  Click on the notebook file (ending in `.ipynb`).
3.  Click the **"Open in Colab"** badge at the top of the file.
4.  Once Colab opens, click **File > Save a copy in Drive** so you can edit and run the code yourself.

## 🛠️ Required Libraries
For most tutorials, we will rely on the following standard scientific Python stack (pre-installed in Colab):
* `numpy`: For matrix math and differential equations.
* `matplotlib`: For plotting neural activity and decision boundaries.
* `scikit-learn`: For accessing datasets and evaluating models.
